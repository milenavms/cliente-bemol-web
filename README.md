### Overview

 Comunica com a mesma API.
 
 ### Funcionalidade
 - Cadastro de cliente
 - Login de cliente 
 
 ### Detalhes
 
 - A aplicação foi desenvolvida em Reactjs 
 - As chamada de  api foram feitas com o fecth
 - Estilização com Bootstrap
 
  ### Executando
 
 - Baixar do projeto
 - Abrir com o projeto
 - Baixar as dependências
 ```
npm install
```
 - Start
 ```
npm start
```
